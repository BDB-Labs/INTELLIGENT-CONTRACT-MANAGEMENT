Public Demo

ICM Contract Intelligence

A sanitized public demo of the contract-intelligence workflow for public-infrastructure and roadway agreements. These examples use curated sample packages and show intake analysis, committed-state carry-forward, obligations, and monitoring posture without exposing internal-only context.

Explore demo cases View repository

What this demo shows

  • Perspective-aware intake review
  • Clause and risk findings
  • Committed obligations
  • Monitoring and alert posture
  • Sanitized external reporting

How to read this demo

Each case below links to a generated external dashboard built from a curated sample contract package. The dashboards are shareable artifacts, not internal operator views.

What is omitted

Internal-only context, local paths, private review actions, and strategy-only notes are removed from these artifacts by design.

Reference runtime

The real Python/FastAPI reference implementation is intended to run separately on Render while this Vercel surface stays public and read-only.
