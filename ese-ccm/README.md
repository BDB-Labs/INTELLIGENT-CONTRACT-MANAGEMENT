# ESE-CCM v1.0 – Construction Contract Compliance Manager Pack

This pack runs on top of the existing ese-cli framework.

Quick Start:
ese run tests/test_contract.txt --pack ese-ccm --mode full --with-context --with-budget

Outputs:
- external_report.md (clean Colleen-style report)
- internal_trace.json (full traceability)
- obligation_register.json (post-award ready)
